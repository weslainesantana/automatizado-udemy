# Projeto de Automação Web - Udemy

Projeto de automação de testes para o site da Udemy usando Selenium WebDriver e JUnit.

## Tecnologias
- Java
- Selenium WebDriver
- JUnit 4
- Maven
- WebDriverManager

## Funcionalidades
- Pesquisa de cursos na Udemy
- Verificação de resultados
- Page Object Pattern

## Como executar
```bash
mvn test
